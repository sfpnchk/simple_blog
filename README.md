# Setup project

## Setup dependencies

1. Install python 3.8 and dev tools
2. Install project dependencies `make setup`
3. Fill next DB credentials in env variables \n
   `DB_NAME
   DB_USER
   DB_PASSWORD
   DB_HOST
   DB_PORT`
4. Setup db schema `make migrate`

# Run project
`make run`
